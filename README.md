## Summary

A [Wordpress](http://wordpress.org/) Theme using [Bootstrap](http://twitter.github.io/bootstrap/), [Font Awesome](http://fortawesome.github.io/Font-Awesome/) and Custom Wordpress Functions and Flat Elements.

## Usage

### Features

1. Carousel
2. Slides Post Type
3. Contact Form
4. Sidebar Widgets
5. Theme Options
6. Breadcrumbs
7. Blog

### Theme Options

* Breadcrumbs
* Google Analytics
* Contact Information
* Social Media Links

### Page Templates

* Blog
* Home
* Full-Width
* Parent-Page
* Child-Page
* Sitemap
* Projects
* Contact
* Gallery
